# Future Spyro Reignited Trilogy Workspace with Dummy C++ Classes already defined.

------

## What is and what is not this project is:

- This project is not an attempt to reimplement or reverse engineer the game.
- This project aims to create non-functioning classes and blueprints to help anyone who wants to mod the game.
- This project also hosts the files for the Mod Loader. 



## How can I use this project?

You require **Unreal Engine 4.19.2** and also **Visual Studio 2017** with G**ame development with C++ packages**, Windows 8.1 SDK installed.
